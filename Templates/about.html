About ClassiNews

Understanding how our AI-powered news classifier works

What is ClassiNews?

ClassiNews is an intelligent news classification system that automatically categorizes news articles into predefined categories using advanced Natural Language Processing (NLP) and Machine Learning techniques.

Our system can instantly analyze headlines and full articles, determining whether they belong to categories like World News, Sports, Business, or Science & Technology.

How It Works

1

Text Preprocessing

Input text is cleaned, lowercased, and stopwords are removed to prepare for analysis.

2

Feature Extraction

TF-IDF (Term Frequency-Inverse Document Frequency) vectorization converts text into numerical features.

3

Classification

Our trained Multinomial Naive Bayes model predicts the most likely category based on learned patterns.

Technology Stack

Backend: Python, Flask
ML Framework: scikit-learn
NLP: NLTK, TF-IDF Vectorization
Model: Multinomial Naive Bayes
Frontend: HTML5, CSS3, JavaScript
Deployment: Flask Web Server
